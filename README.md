# Loyalz Weekly Tracker

Proyecto listo para deployar en Vercel.

## Ejecutar local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Subí esta carpeta a un repositorio en GitHub.
2. En Vercel, elegí **Add New > Project**.
3. Importá el repo.
4. Framework preset: **Vite**.
5. Build command: `npm run build`
6. Output directory: `dist`
7. Deploy.

## Dependencias

- react
- react-dom
- recharts
- papaparse
- xlsx
- lucide-react
- tailwindcss
